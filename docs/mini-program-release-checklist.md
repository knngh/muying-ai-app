# 小程序上线前抽检清单

## 核心路径
- 首页：不登录可进入权威知识库和孕周日历；不出现问题助手入口。
- 知识库：搜索、阶段筛选、机构筛选、场景卡片、加载更多都可用。
- 知识详情：标题、摘要、正文分段、来源机构、来源更新时间、同步时间展示正常。
- 中文辅助阅读：英文来源可进入详情并准备中文阅读版；中文来源不展示无意义翻译入口。
- 来源原文：有效机构原文链接可打开；无效链接显示不可用提示。
- 孕周日历：不登录可浏览，登录后可保存待办和记录。
- 孕期档案：未登录跳转登录；登录后可查看阶段信息和记录摘要。
- 我的：登录、退出登录、编辑资料、内容可信说明入口可用。

## 内容可信与合规
- 知识库页和我的页可进入“内容可信说明”。
- 内容可信说明页包含来源范围、中文辅助阅读规则、高风险就医提示、隐私与使用范围。
- 小程序内不出现“问题助手”主入口。
- 小程序不要求上传身份证件、真实姓名或其他敏感身份材料。
- 高风险文章详情页会展示“高风险提示”卡，例如出血、腹痛、破水、胎动异常、高热、黄疸、用药或治疗方案。
- 任何涉及诊断、用药、治疗方案的文案都明确不替代医生判断。

## 知识库抽样
- 抽查 10 篇中国权威源文章：标题、摘要、正文、来源链接、时间字段正常。
- 抽查 10 篇国际权威源文章：中文导读、中文辅助阅读、原文切换正常。
- 抽查 5 篇高风险主题文章：风险提示卡出现，提示语不过度诊断。
- 抽查 5 篇无正文或正文较短文章：不出现空白页，摘要模式提示正常。
- 抽查 5 篇泛化标题文章：列表页不直接显示 Resources、Guide、Article 等低价值标题。

## 技术验收
- `npm run type-check`
- `npm run build:mp-weixin`
- `npm run smoke:mp-weixin`，若 WeChat DevTools 提示重新登录，记录为本机环境问题。

## 发布备注
- 当前小程序定位为权威知识库、孕周日历和基础档案。
- 咨询式连续追问能力暂不在小程序上线。
- App 端可继续承接深度 AI 阅读整理和行动卡能力。
