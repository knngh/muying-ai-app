# 微信小程序使用宝塔部署的 H5 教程

本文档介绍如何在微信小程序中使用已在宝塔部署的 H5 应用。

---

## 方式一：通过 web-view 嵌入 H5 页面（推荐）

### 步骤 1：配置业务域名

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入 **设置与开发** → **开发管理** → **开发设置**
3. 找到 **业务域名**，点击 **设置**
4. 添加你的 H5 网站域名（如 `h5.xxx.com`）
5. 下载验证文件并上传到宝塔网站根目录
6. 确认验证成功

> ⚠️ 注意：必须使用 HTTPS 域名

### 步骤 2：在小程序中添加 web-view 页面

在 `src/pages/` 下创建 `webview` 目录，添加 `webview.vue`：

```vue
<template>
  <web-view :src="url"></web-view>
</template>

<script>
export default {
  data() {
    return {
      // 替换为你的 H5 部署地址
      url: 'https://h5.xxx.com'
    }
  },
  onLoad(options) {
    // 支持从外部传入 URL 参数
    if (options.src) {
      this.url = decodeURIComponent(options.src)
    }
  }
}
</script>
```

### 步骤 3：配置路由

在 `src/pages.json` 中添加页面配置：

```json
{
  "path": "pages/webview/webview",
  "style": {
    "navigationBarTitleText": "H5 应用",
    "navigationBarBackgroundColor": "#ffffff",
    "navigationBarTextStyle": "black"
  }
}
```

### 步骤 4：从其他页面跳转

在你需要打开 H5 页面的小程序页面中：

```javascript
// 跳转到 web-view 页面
uni.navigateTo({
  url: '/pages/webview/webview?src=' + encodeURIComponent('https://h5.xxx.com')
})
```

## 方式二：通过 URL Scheme 拉起 H5

适用于需要在外链浏览器中打开 H5 页面的场景。

### 生成 URL Scheme

在 H5 页面中添加跳转按钮，使用微信小程序的 URL Scheme：

```javascript
// 在 H5 页面中
const scheme = 'weixin://dl/business/?t=你的小程序页面路径'
// 或者生成小程序码
```

### 在小程序中生成 scheme（需要后端 API）

调用微信接口生成 URL Scheme：

```
GET https://api.weixin.qq.com/wxa/geturlscheme?access_token=ACCESS_TOKEN
```

请求体：
```json
{
  "jump_wxa": {
    "path": "pages/index/index",
    "query": ""
  },
  "expire_type": 1,
  "expire_time": 1600000000
}
```

## 常见问题

### 1. web-view 不显示内容

- 检查是否配置了业务域名
- 检查是否使用 HTTPS
- 检查域名是否已备案（国内服务器需要）

### 2. 页面显示白屏

- 在微信开发者工具中检查控制台错误
- 确认 H5 页面在手机浏览器中可正常访问

### 3. 登录态如何同步

- 如果 H5 需要登录，需要与小程序的登录态打通
- 可以使用 openid 绑定用户身份

### 4. 如何传递参数给 H5

通过 URL Query 参数：

```javascript
// 小程序
uni.navigateTo({
  url: '/pages/webview/webview?src=' + encodeURIComponent('https://h5.xxx.com?token=xxx')
})

// H5 页面获取
const params = new URLSearchParams(window.location.search)
const token = params.get('token')
```

## 下一步建议

如果你需要实现真正的后端功能（如 AI 对话、用户登录等），需要：

1. **部署后端服务**：在宝塔上部署 Node.js/Python/Java 后端
2. **修改前端 API**：将 `src/api/` 目录下的 mock 数据替换为真实 API 调用
3. **配置后端域名**：在小程序后台配置 request 合法域名
